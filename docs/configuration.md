# Configuration

## Requirements

- Node.js `20+`
- npm
- Playwright browser dependencies for live session extraction

Install:

```bash
npm install
```

## Environment Variables

Create `.env` in this JS project root:

- `LINKEDIN_EMAIL` or `LINKEDIN_USERNAME`
- `LINKEDIN_PASSWORD`
- `LI_COOKIE` generated by the session extractor
- `LI_CSRF_TOKEN` generated by the session extractor

Minimal initial `.env`:

```env
LINKEDIN_EMAIL=your_email
LINKEDIN_PASSWORD=your_password
```

Then run:

```bash
node module/session-extractor.js --test
```

## Output Paths

- `output/results/`
- `output/progress/`
- `output/session/`

## Test Fixture Overrides

These are used by the automated tests and can also be used for local dry runs:

- `LINKEDIN_SCRAPER_MOCK_SEARCH_RESPONSE_FILE`
- `LINKEDIN_SCRAPER_MOCK_GEO_RESPONSE_FILE`
- `LINKEDIN_SCRAPER_DISABLE_SLEEP=1`
- `LINKEDIN_SCRAPER_NONINTERACTIVE=1`
